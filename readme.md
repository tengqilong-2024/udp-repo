# 5米休闲船嵌入式控制系统



## 基本思想

使用UDP通讯，包括串口服务器，GNSS高精度定位仪，逸动左右推进器和云卓遥控器。

控制系统功能是，上反馈定位信息和推进器转速给UI，上传输定位信息和推进器信息给算法计算机；下控制电机和解析数据。

串口服务器地址：192.168.2.100

​               GNSS端口：31001  <--->29002

​              推进器左：31002 <--->29002

​              推进器右：31003 <--->29002

​              遥控器   ：31004  ---->29002
